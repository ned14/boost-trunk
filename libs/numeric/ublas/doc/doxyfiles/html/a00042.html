template<class T, std::size_t N>
class boost::numeric::ublas::bounded_vector< T, N >

a dense vector of values of type T, of variable size but with maximum . A dense vector of values of type T, of variable size but with maximum . The default constructor creates the vector with size . Elements are constructed by the storage type bounded_array, which need not initialise their value.
