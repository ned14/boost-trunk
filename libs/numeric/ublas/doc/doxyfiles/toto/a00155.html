Detailed Description

template<class T, class ALLOC>
class boost::numeric::ublas::identity_matrix< T, ALLOC >

An identity matrix with values of type T.

Elements or cordinates are equal to 1 (one) and all others to 0 (zero). Changing values does not affect the matrix, however assigning it to a normal matrix will make the matrix equal to an identity matrix. All accesses are constant du to the trivial values.

Template Parameters:
Tthe type of object stored in the matrix (like double, float, complex, etc...)
ALLOCan allocator for storing the zeros and one elements. By default, a standar allocator is used.
