Detailed Description

template<class M, class TRI>
class boost::numeric::ublas::hermitian_adaptor< M, TRI >

A Hermitian matrix adaptator: convert a any matrix into a Hermitian matrix expression.

For a -dimensional matrix, the hermitian_adaptor will provide a hermitian matrix. Storage and location are based on those of the underlying matrix. This is important because a hermitian_adaptor does not copy the matrix data to a new place. Therefore, modifying values in a hermitian_adaptor matrix will also modify the underlying matrix too.

Template Parameters:
Mthe type of matrix used to generate a hermitian matrix
