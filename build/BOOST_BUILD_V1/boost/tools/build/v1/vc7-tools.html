Boost.Build

vc7 toolset

Introduction

Boost.Build's vc7 toolset supports the Microsoft Visual C++ .NET tools. This toolset is an extention to the msvc toolset and is usefull for when you need to have access to both version 6 and version 7 of the toolsets.

Configuration Variables

The vc7 toolset responds to the following variables which can be set in the environment or configured on the jam command-line using -sVARIABLE_NAME=value:
Variable Name Semantics Default Notes
VC7_ROOT Path to installation of VC7. C:\Program Files\Microsoft Visual Studio .NET\VC7 If MSVCDir is set in the environment, the toolset will assume VCVARS32.BAThas already been called, and will ignore this variable.
