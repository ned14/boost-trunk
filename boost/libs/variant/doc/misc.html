Compatibility

To date the code and test harness have been compiled and tested successfully using Microsoft Visual C++ 7.0 and GNU C++ 2.95/3.1.

Instantiation with a boost::mpl sequence is not supported under MSVC.

Troubleshooting

  • Compiler: GNU C++. Error message:

    variant.hpp: template instantiation depth exceeds maximum of 17
    variant.hpp: (use -ftemplate-depth-NN to increase the maximum)

    Solution: Passing -ftemplate-depth-50 to the compiler. This will increase the maximum depth to 50.


  • Compiler: MSVC7. Error message:

    boost\mpl\clear.hpp(36) : fatal error C1076: compiler limit : internal heap limit reached;
    use /Zm to specify a higher limit

    Solution: Passing /Zm200 to the compiler. This will increase the heap by a factor of 200%.
