#define BOOST_PP_IF(COND,THEN,ELSE)

Expands to THEN if COND != 0 and ELSE if COND == 0.

COND must expand to an integer literal in the range [0, BOOST_PP_LIMIT_MAG].

For example, BOOST_PP_IF(0,1,2) expands to 2.
