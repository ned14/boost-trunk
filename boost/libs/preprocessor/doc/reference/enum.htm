#include <boost/preprocessor/enum.hpp>

#define BOOST_PP_ENUM(N,F,P)

Generates a comma separated list.

In other words, expands to the sequence:

  F(0,P), F(1,P), ..., F(N-1,P)

Uses

  • BOOST_PP_REPEAT()
