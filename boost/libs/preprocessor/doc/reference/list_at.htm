#include <boost/preprocessor/list/at.hpp>

#define BOOST_PP_LIST_AT(L,I)

Expands to the I:th element of the list L. The first element is at index 0.

For example,

  BOOST_PP_LIST_AT(BOOST_PP_TUPLE_TO_LIST(3,(A,B,C)),1)

expands to B.

Uses

  • BOOST_PP_WHILE()
