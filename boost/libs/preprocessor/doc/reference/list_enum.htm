#include <boost/preprocessor/list/enum.hpp>

#define BOOST_PP_LIST_ENUM(L)

Converts the list to a comma separated list.

For example,

  BOOST_PP_LIST_ENUM(BOOST_PP_TUPLE_TO_LIST(3,(A,B,C)))

expands to:

  A,B,C

Uses

  • BOOST_PP_FOR()
