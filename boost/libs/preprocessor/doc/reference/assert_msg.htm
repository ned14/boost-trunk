#include <boost/preprocessor/assert_msg.hpp>

#define BOOST_PP_ASSERT_MSG(C,MSG)

Expands to nothing if C != 0 and to MSG if C == 0.

#define BOOST_PREPROCESSOR_ASSERT_MSG(C,MSG)

Obsolete. Use BOOST_PP_ASSERT_MSG().
