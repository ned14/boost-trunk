The BOOST_PP_SET_FOR_EACH macro repeats a macro for each element in a set.

Usage

BOOST_PP_SET_FOR_EACH(macro, data, set)

Arguments

macro
A ternary macro of the form macro(r, data, elem).  This macro is expanded by BOOST_PP_SET_FOR_EACH with each element in set.  It is expanded with the next available BOOST_PP_FOR repetition, the auxiliary data, and the current element.
data
Auxiliary data passed to macro.
set
The set for which macro will be invoked on each element.

Remarks

This macro is a repetition construct.  If set is (a)(b)(c), it expands to the sequence:
macro(r, data, a) macro(r, data, b) macro(r, data, c)
For maximum efficiency, use BOOST_PP_SET_FOR_EACH_R.

See Also

  • BOOST_PP_SET_FOR_EACH_R

Requirements

Header:  <boost/preprocessor/set/for_each.hpp>

Sample Code

#include <boost/preprocessor/cat.hpp>
#include <boost/preprocessor/set/for_each.hpp>

#define SET (w)(x)(y)(z)

#define MACRO(r, data, elem) BOOST_PP_CAT(elem, data)

BOOST_PP_SET_FOR_EACH(MACRO, _, SET) // expands to w_ x_ y_ z_
