Filesystem Do-list

Should directory_iterator ignore these files? Later: this is the is_accessible() versus exists() problem. Modify the sample program accordingly. Still Later: use status() or symlink_status().
     
  • Windows: //network-name style paths need more analysis and test cases for system_complete() and complete(). Also, path_test cases at line 410 need review, correction.
     
  • Add a path::swap member function guaranteed not to throw. (Geurt Vos).
     
  • Add hard link support.
