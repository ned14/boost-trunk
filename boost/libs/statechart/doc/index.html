Overview

Welcome to boost::fsm, a C++ library for finite state machines. Features include:

  • Straightforward transformation from UML state chart to executable C++ code and vice versa (no code generator required)
  • Comprehensive UML semantics support (history not yet implemented)
    • Hierarchical (composite, nested) states
    • Orthogonal (concurrent) states
    • Entry-, exit- and transition-actions
    • Guards
  • Error handling support
  • Full type-safety
  • State-local storage
  • Customizable resource acquisition

The library is not yet complete. The following is still lacking:

  • Support for state machines running in their own threads
  • Shallow/deep history
  • Full documentation (currently there is only a tutorial, a rationale and a FAQ page)

This is a preliminary submission to encourage feedback. All comments are most welcome!
