Acknowledgments

Very special thanks go to:

  • Aleksey Gurtovoy, the developer of the ingenious meta programming library (boost::mpl). The interface as well as the implementation of boost::fsm hugely benefit from Alekseys work. I would have given up long ago without mpl. Moreover, Aleksey's double dispatch implementation in his FSM framework gave me fresh ideas after I had come to the conclusion that my dynamic_cast-based solution was too bloody bad ;-)

Special thanks go to:

  • Peter Dimov for developing two other important libraries (lightweight_mutex, intrusive_ptr) boost::fsm is building on
  • the countless folks who developed important boost infrastructure like config, regression tests, build, html templates, etc.
  • Mitsuo Fukasawa, who is the first person using boost::fsm in a real-world project. He gave invaluable feedback and convinced me to make history a top priority

Thanks for feedback and/or encouragement go to:
