  • Boost 1.33 release

Boost 1.33 release

  • Added serialization support for Boost.Serialization.
  • Internal changes aimed at reducing the length of symbol names generated by the compiler. This results in shorter and more readable error messages and has also a beneficial impact on compilers with strict limits on symbol name lengths.
  • Maintenance fixes.
