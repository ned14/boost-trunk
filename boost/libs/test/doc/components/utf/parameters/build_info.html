The 'build info' parameter

Parameter Name: Print build info
Environment variable name: BOOST_TEST_BUILD_INFO
Command line argument name: build_info
Acceptable Values: no
yes
Description: makes the framework to print build information that include: platform, compiler, STL implementation in use and boost version.
