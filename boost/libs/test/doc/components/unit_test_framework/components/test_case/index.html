The Test Case

The Unit Test Framework provides an ability for the user to create the test case based on many different sources.

  • test case interface - the abstract interface all specific test cases should comply to. Could be used for framework extensions
  • function_test_case - zero arity free function based test case
  • class_test_case - zero arity class member function based test case
  • parameterized_function_test_case - arity one free function based test case
  • parameterized_class_test_case - arity one class member function based test case
  • boost_function_test_case - zero arity boost::function based test case
  • parameterized_boost_function_test_case - arity one boost::function based test case
