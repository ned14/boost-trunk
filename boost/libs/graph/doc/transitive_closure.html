transitive_closure 

template <class Graph>
void transitive_closure(Graph& g)
The transitive closure of a graph G = (V,E) is a graph G' = (V,E') such that E' contains an edge (u,v) if and only if G contains a path from u to v. The transitive_closure() function transforms the input graph g into the transitive closure of itself.

IN/OUT: Graph& g
A directed graph, where the Graph type must model the Vertex List Graph and Edge Mutable Graph concepts.
