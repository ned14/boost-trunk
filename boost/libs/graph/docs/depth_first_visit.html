depth_first_visit

Graphs: directed and undirected
Properties: color
Complexity: time: O(E)
Where Defined: boost/graph/depth_first_search.hpp 

template <class IncidenceGraph, class DFSVisitor, class ColorPMap>
void depth_first_visit(IncidenceGraph& g,
	               typename graph_traits<IncidenceGraph>::vertex_descriptor u, 
	               DFSVisitor& vis, ColorPMap color)

This function is the recursive part of the depth-first search. The main purpose of the function is for the implementation of depth_first_search() though sometimes it is useful on its own.

Requirements on Types

  • The type IncidenceGraph must be a model of IncidenceGraph.
  • The type DFSVisitor must be a model of DFSVisitor.
  • The type ColorPMap must be a model of ReadWritePropertyMap, and the value type of Color must be a model of ColorValue.
