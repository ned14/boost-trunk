depth_first_visit 

template <class IncidenceGraph, class Vertex, class Visitor>
void depth_first_visit(IncidenceGraph& G, Vertex u, Visitor& v);

This function is the recursive part of the depth-first search. The main purpose of the function is for the implementation of depth_first_search() though sometimes it is useful on its own. The responsibility for coloring, etc. is left to the visitor. The visitor's explore() function returns bool which determines whether the target vertex will be explored (when it returns true) or not. The implementation of depth_first_search() uses the dfs_visitor in conjunction with this depth_first_visit() function.

Where Defined

boost/graph/depth_first_search.hpp

Requirements on Types

  • The type IncidenceGraph must be a model of IncidenceGraph.
  • The type Vertex must be convertible to type boost::graph_traits<IncidenceGraph>::vertex_descriptor.
  • The type Visitor must be a model of Visitor.
