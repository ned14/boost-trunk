ForwardIterator

Description

A Forward Iterator is a TrivialIterator that provides a way to obtain the next iterator in a sequence.

Definitions

A forward iterator is past-the-end if it points beyond the last element of a sequence. Past-the-end iterators are nondereferenceable.
A forward iterator is valid if it is dereferenceable or past-the-end.
A forward iterator i is incrementable if there is a "next" iterator, that is, if i::next expression is well-defined. Past-the-end iterators are not incrementable.
A forward iterator j is reachable from a forward iterator i if, after recursive application of next operation to i a finite number of times, i is identical to j. [1]
The notation [i,j) refers to a range of iterators beginning with i and up to but not including j.
The range [i,j) is a valid range if both i and j are valid iterators, and j is reachable from i [2].
