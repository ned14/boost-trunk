TrivialIterator

Description

A TrivialIterator X is a type that represents a reference to an element of some Sequence, and allows to access the element through its nested type member [1]. There is no guarantee that TrivialIterator supports any traversal operations. (Specific types of iterators, such as ForwardIterator do provide such guarantee).

Definitions

A TrivialIterator can be dereferenceable, meaning that accessing its type member is a well-defined operation.
