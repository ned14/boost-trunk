Predefined function classes

Introducation

The boost::mpl library includes many different predefined function classes, including [arithmetic operations]?, comparisons?, and [logical operations]?. It is possible to perform very sophisticated operations without actually writing a new function class, simply by combining predefined function classes, using either ?? or [expression templates]?.

Example

Find the product of elements in a value_list?. 

namespace mpl = boost::mpl;


typedef mpl::value_list?<5,7,1,4,2> values;
typedef mpl::accumulate?< values, mpl::int_c?<1>, mpl::multiplies?<mpl::_1?> >::type result;


BOOST_STATIC_ASSERT(result::value == 280);
