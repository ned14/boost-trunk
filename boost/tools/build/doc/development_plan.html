Boost Build Development Plan

  • Milestone 2 (October 25, 2002)

    Boost buildable with gcc and also usable as part of another project.

  • Milestone 3 (April 24, 2003)

    Two gcc versions and two other compilers are supported. Some other tools (e.g. lex and BoostBook) are implemented.

  • Milestone 4 (TBD)

    Complete testing framework for Boost.

  • Alpha (TBD)

    Optimization. Using buildable tools: consider using bison, which is itself build with Boost.Build.

  • Beta (TBD)

    Feature-complete milestone.

  • 2.0 Release (TBD)

    Bugfixes
