Function hash_value

boost::hash_value —

Synopsis

template<typename T, typename A> 
  std::size_t hash_value(std::deque<T, A> const & val);

Description

Returns: hash_range(val.begin(), val.end());
Throws: Only throws if hash_value(T) throws.
Notes: This is an extension to TR1
