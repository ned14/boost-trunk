boost::swap —

Synopsis

  void swap(variant<T1, T2, ..., TN> & lhs, variant<T1, T2, ..., TN> & rhs);

Description

Effects: Swaps lhs with rhs by application of variant::swap.
Throws: May fail with any exception arising from variant::swap.
